# Git Integration

**Domain**: Git version control workflows

---

## Git Tools

<tools>
  <neogit>
    - Magit-like Git interface
    - Staging/committing
    - Branch management
    - Interactive rebasing
  </neogit>
  
  <gitsigns>
    - Inline diff markers
    - Hunk navigation
    - Stage/unstage hunks
    - Blame annotations
  </gitsigns>
  
  <diffview>
    - File history viewing
    - Diff comparison
    - Merge conflict resolution
    - PR review
  </diffview>
  
  <lazygit>
    - Terminal UI for Git
    - Full Git operations
    - Visual branching
  </lazygit>
</tools>

---

## Worktree Support

<worktrees>
  - Multiple working directories
  - Branch isolation
  - Parallel development
  - git-worktree.nvim plugin
</worktrees>

---

## Common Workflows

<workflows>
  <commit>
    - Stage changes (Neogit or gitsigns)
    - Write commit message
    - Push to remote
  </commit>
  
  <review>
    - View diffs (diffview)
    - Navigate hunks (gitsigns)
    - Blame inspection
    - History exploration
  </review>
</workflows>
