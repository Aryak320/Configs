# Implementation Plan Template

Template for implementation plans.

```markdown
# Implementation Plan: {Project Name}

**Version**: v1
**Status**: [NOT STARTED]
**Created**: YYYY-MM-DD
**Based On**: Research OVERVIEW at {path}

## Overview

{Summary of what will be implemented}

## Goals

- {Goal 1}
- {Goal 2}

## Phases

### Wave 1 (Parallel Execution)

#### Phase 1: {Phase Name}
- **Dependencies**: None
- **Complexity**: Low/Medium/High
- **Estimated Time**: X hours
- **Tasks**:
  1. {Task 1}
  2. {Task 2}
- **Validation**: {How to verify success}

#### Phase 2: {Phase Name}
- **Dependencies**: None
- **Complexity**: Low/Medium/High
- **Tasks**: ...

### Wave 2

#### Phase 3: {Phase Name}
- **Dependencies**: Phase 1, Phase 2
- **Tasks**: ...

## Testing Strategy

{How implementation will be tested}

## Rollback Plan

{How to undo changes if needed}

## Metadata

- **Total Phases**: {N}
- **Total Waves**: {N}
- **Estimated Duration**: {X} hours
```
